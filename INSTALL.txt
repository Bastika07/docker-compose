# LanCache Monitoring - Installation

## Schnellstart:
1. ZIP-Archiv entpacken
2. In das Verzeichnis wechseln: `cd lancache-monitoring`
3. .env konfigurieren: `cp .env.example .env` und anpassen
4. Setup ausführen: `chmod +x setup_monitoring.sh && ./setup_monitoring.sh`
5. System starten: `docker-compose up -d`

## Zugriff:
- Grafana: http://localhost:3000 (admin/admin123)
- Prometheus: http://localhost:9090
- Quick Stats: http://localhost:8080
- Metriken: http://localhost:9114/metrics

## Wichtige .env Einstellungen:
```
LANCACHE_IP=10.0.0.20          # Ihre LanCache IP
DNS_BIND_IP=10.0.0.20          # DNS IP (meist identisch)
CACHE_ROOT=./lancache          # Cache-Speicherort
CACHE_DISK_SIZE=1000g          # Max. Cache-Größe
UPSTREAM_DNS=8.8.8.8 8.8.4.4  # DNS-Server
GRAFANA_PASSWORD=admin123      # Grafana Passwort
```

## Fehlerbehebung:
- Logs: `docker-compose logs -f`
- Neustart: `docker-compose restart`
- Reset: `docker-compose down && docker-compose up -d`

Für detaillierte Informationen siehe README.md
